# Laba2
ЛАБОРАТОРНАЯ РАБОТА №2 по дисциплине: «Программирование»
на тему: «Классификация языков программирования»
Вариант 6

Цели и задачи работы: изучение функций ввода-вывода данных, программирования вычисления значения выражения. Формирование практических навыков реализации алгоритмов на различных языках программирования, сравнительный анализ синтаксических и семантических особенностей языков

Задание к работе: 
1.	реализовать линейный вычислительный процесс. Самостоятельно решить задачи в соответствии с индивидуальным вариантом. Реализовать представленные задачи на языках программирования C++, Go (Golang), Rust, Assembler, Kotlin, JavaScript, Swift, Python, Java, Elixir.
2.	провести отладку и тестирование программ на различных входных данных
3.	проанализировать различия в объёме кода, читаемости и особенностях реализации одного алгоритма на разных языках программирования.

Методика выполнения работы:
1.	разработать алгоритм решения задачи по индивидуальному заданию;
2.	выбрать стек языков программирования согласно требованиям;
3.	написать и отладить программу с вводом-выводом информации с консоли и из файла;
4.	протестировать работу программы на различных исходных данных;
5.	изменить формат вывода, проверить работу программы при другом формате вывода.

Задание №1
Условие задачи: «6. (Кооперативы): В частном секторе расположены N подряд стоящих вышек, обеспечивающих сотовую связь. Вам дается строка S длины N, где S[i] = '1' означает, что вышка работает и обеспечивает кооператив связью, S[i] = '0' означает отсутствие связи. Вышка имеет показатель X, который означает, сколько кооперативов слева и справа от себя она может обеспечивать связью. Вам нужно выяснить, могут ли все кооперативы пользоваться сотовой связью. 
Пример 1: N = 4, X = 1, S = “1001”, Результат: Yes 
Пример 2: N = 4, X = 0, S = “1001”, Результат: No 
Пример 3: N = 5, X = 1, S = “10001”, Результат: No».

Задание №2
Условие задачи: «6. (Секретный агент): Агент разведки перехватил последовательность радиосигналов, N – их количество. Каждый сигнал это целое положительное число (частота в герцах). По характеру изменения частот нужно определить тип сообщения: “АТАКА” – если частоты строго возрастают (каждая следующая больше предыдущей); “ОТСТУПЛЕНИЕ” – если частоты строго убывают (каждая следующая меньше предыдущей); “ВСТРЕЧА” – если частоты чередуются (то возрастают, то убывают: первая < вторая > третья < четвёртая ИЛИ первая > вторая < третья > четвёртая); “ШУМ” – если ни один из предыдущих паттернов не подходит.
